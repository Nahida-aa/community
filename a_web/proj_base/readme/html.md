## body

body标签：这是HTML文档的主体部分，通常包含了用户在浏览器中可以看到的所有内容，如文本、图片、链接、表格、列表等。

```html
    <body>
        <header>
            <!-- 网站标题、导航菜单等 -->
            <nav>
                <!-- left top -->
                <button>
                    <span class="navbar-toggler-icon"></span>
                </button>
                <ul>
                    <li><a href="#home">首页</a></li>
                    <li><a href="#about">关于我们</a></li>
                    <li><a href="#contact">联系我们</a></li>
                </ul>
            </nav>
            <h1>我的网站</h1>
        </header>
        <main>
            <!-- 网站主要内容 -->
            <section id="home">
                <h2>欢迎来到我的网站</h2>
                <p>这是一段介绍文本。</p>
            </section>
            <section id="about">
                <h2>关于我们</h2>
                <p>这是一段关于我们的文本。</p>
            </section>
            <section id="contact">
                <h2>联系我们</h2>
                <p>这是一段联系我们的文本。</p>
            </section>
        </main>
        <footer>
            <!-- 网站底部信息，如版权声明、联系方式等 -->
            <p>版权所有 &copy; 2022 我的网站</p>
        </footer>
    </body>
```
